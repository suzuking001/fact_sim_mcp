# Hello MCP (Minimal)

This is a minimal MCP server with exactly one tool: `ping`.

- Input: none
- Output: `"pong"` and the current server time (ISO 8601)
- Error handling: tool-level, startup-level, process-level
- Logging: stderr only, includes request ID for tool calls

## File Structure

```text
fact_sim_mcp/
  src/
    index.ts
  dist/                    # generated by npm run build
  package.json
  tsconfig.json
  mcp.config.example.json
  .gitignore
  README.md
```

## Requirements

- Node.js 18+

## Run

```bash
npm install
npm run build
npm start
```

## MCP Client Config (VSCode/Codex)

Configure your MCP client with `command` and `args`.  
Example from `mcp.config.example.json`:

```json
{
  "mcpServers": {
    "hello-mcp": {
      "command": "node",
      "args": ["C:/path/to/fact_sim_mcp/dist/index.js"]
    }
  }
}
```

Replace the path with your actual absolute path.
